# GoodSkinGal

A beautiful, responsive landing page for **GoodSkinGal** — Kristin's personal esthetics business.

Built with React + Vite. Open `index.html` with Cursor Live Preview, or run the dev server locally.

## Quick Start

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).

For **Cursor Live Preview**: right-click `index.html` → **Open with Live Preview** (requires `npm install` first so Vite can serve the React app).

## Sections

- **Home** — Hero with soft animated background
- **About** — Meet Kristin
- **Gallery** — Photo grid with lightbox (placeholder gradients until real photos are added)
- **Forms** — Client intake + consent/waiver
- **Contact** — Message form + Instagram [@goodskingal](https://instagram.com/goodskingal)
