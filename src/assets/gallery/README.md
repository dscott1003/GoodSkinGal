# Gallery photos

Drop your photos into this folder (`src/assets/gallery/`).

- Supported: `.jpg`, `.jpeg`, `.png`, `.webp`
- They are picked up **automatically** and shown in the crossfading
  slideshow on the site — no code changes needed.
- Files are shown in **alphabetical order**, so prefix them if you want a
  specific sequence, e.g. `01_studio.jpg`, `02_facial.jpg`, ...
- Keep each image reasonably sized (long edge ~1600px, optimized JPEG)
  so the site stays fast.

Once at least one image is here, the placeholder grid is replaced by the
slideshow automatically.
